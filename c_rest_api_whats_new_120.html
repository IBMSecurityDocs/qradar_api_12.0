What's new in REST API Version 12.0

IBM® QRadar® V7.3.3 introduces version 12.0 of the API endpoints.

New endpoints

RESTful API 12.0 introduces new categories of API endpoints and updates to existing endpoints in the following categories:

Configuration endpoints
Backup and restore
Event sources
Extension management
Staged configuration endpoints
Backup and restore
System endpoints
Email servers

Deprecated endpoints

All version 10.x API endpoints are marked as deprecated in QRadar V7.3.3.

Although deprecated endpoints continue to function, they will be removed in a future release. You must update your integration to use the most recent version of the QRadar RESTful API. Responses to deprecated endpoint requests include a Deprecated response header.
